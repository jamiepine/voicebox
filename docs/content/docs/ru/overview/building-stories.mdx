## С чего начать

Stories Editor особенно хорошо подходит для подкастов, аудиокниг и любых
материалов, где говорят несколько спикеров.

<Steps>
  <Step title="Создайте историю">
    Откройте **Stories** → **+ New Story**.
  </Step>
  <Step title="Добавьте дорожки">
    Создайте отдельные треки для каждого спикера.
  </Step>
  <Step title="Наполните их клипами">
    Сгенерируйте новые фрагменты или перетащите уже готовое аудио на дорожки.
  </Step>
  <Step title="Расставьте и подрежьте">
    Разложите клипы по таймлайну и при необходимости подрежьте края.
  </Step>
  <Step title="Экспортируйте">
    Соберите финальный аудиофайл.
  </Step>
</Steps>

## Где это особенно полезно

- подкасты с несколькими ведущими
- аудиокниги с рассказчиком и голосами персонажей
- игровые диалоги и сцены
- образовательные материалы с несколькими ролями
